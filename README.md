# Explainability for Crypto Trading  
Interpretable AI methods for understanding Buy/Sell/Hold decisions in volatile crypto markets.

##  Overview
Crypto markets are highly volatile and difficult to interpret. This project applies Explainable AI (XAI) to uncover *why* a trading model makes certain decisions.

##  Goals
- Apply LSTM / RL / Transformer models for crypto prediction  
- Generate explanations using SHAP, Integrated Gradients, and Attention  
- Improve transparency, trust, and risk understanding  


##  Methods Used
- **SHAP** for global + local feature importance  
- **Integrated Gradients** for gradient-based attribution  
- **Attention Visualization** for sequential model inspection  


##  Results
- Buy decisions rely heavily on RSI, MACD, and trend momentum  
- Sell decisions correlate with volatility spikes  
- Explanations align with financial intuition
